randomtreetest
--------------

  This main program does the same as 'treetest', except it creates its own random strings, composed
  of a random amount of random lowercase letters. It will generate 50 of these strings, and then
  make a node to hold each one. These nodes will get passed into the tree, and once fully
  constructed, will be printed in reverse alphabetical order.
